Title : Events - you dont wanna miss
Developed a responsive website that provides real-time information about local events happening in the city. The platform helps users discover nearby cultural programs, workshops, fests, and community activities in one place.
Users can:
Browse upcoming events
View event details (date, time, venue, description)
Stay updated with the latest local happenings
The goal of this project was to make local event discovery simple, accessible, and organized for everyone.

Tech Stack : Frontend based (React)

Installation:
Clone the repository:
git clone https://github.com/<your-username>/projectname.git
cd Events-You dont wanna miss
Install dependencies:
npm install
Start the development server:
npm run dev
